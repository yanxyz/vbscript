# VBScript statements

## 条件语句

If...Then 为单行时可以省略 End If

Select Case

## 循环语句

For...Next 步进循环

For Each...Next 集合循环

While...Wend 没有跳出方法，建议用 Do...Loop

## Exit

退出相关语句

```
Exit Do
Exit For
Exit Sub
Exit Function
Exit Property
```

退出脚本

```
WScript.Quit 1
```
