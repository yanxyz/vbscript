# VBScript

[Wiki](https://en.wikipedia.org/wiki/VBScript)

VBScript 已经停止发展了，不过它还是有一定的用途，有一些其它工具不容易实现的功能。

VBScript 的宿主环境有

- Windows Script Host (WSH)
- Internet Explorer (IE)，IE11 不再支持。
- Internet Information Services (IIS)

本笔记只关注 WSH。
